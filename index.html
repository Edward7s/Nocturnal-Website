    > Nocturnal Premium Options <

    ~> Nocturnal has 4 diffirent types of premium options, all the prices are In the Discord server.
    ~> Normal Tag: Its a custom tag at your choice where u can chose a custom color or a custom gradiant for example having a tag "Just Better" In a dark purple gradiant that goes to light purple or any type It can be rainbow it can be whatever.
    ~> Animated Tag: Its the same as the Normal Tag but its going to write every letter one by one and then delete every letter one by one (This only can be in one color not a gradinat).
    ~> VIP: This will give u a VIP Tag with 4 more extra tags that u can customzie yourself & change them at anytime in game all by yourself, this will also allow u to use the client chat & more features that will get added in the near future.
    ~> High Class: Same as VIP with two more tags and instead of a VIP tag u will get a High Class tag.
    ~> All of those are only if u want to support me.
